
# 排序问题

## 快速排序
```python3
def quick_sort(nums):
    if len(nums) == 0 or len(nums) == 1:
        return nums
    
    mid_value = nums[0]
    below = [x for x in nums[1:] if x <= mid_value]
    above = [x for x in nums[1:] if x > mid_value]

    return quick_sort(below) + [mid_value] + quick_sort(above)
```
